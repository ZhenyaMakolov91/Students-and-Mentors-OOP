# Описание проекта

Данный проект состоит из трех классов: 1) Студенты; 2) Ривьюверы; 3) Лекторы. Функционал проекта позволяет каждому 
классу выполнять определенные функции.

Студенты могут оценивать лекторов, получают оценки от ривьюверов (экспертов, оценивающих домашнюю работу); имеют
параметр - оценки, с помощью которого можно выйчислить среднюю оценку за все домашние задания.
Лекторы получают оценки от студентов; также имеют параметр оценки и возможность вычислить средний балл за лекции. 
Ривьюверы оценивают студентов.

Во всех класса метод __str__ переопределен и по по своему отображает данные для своих экземпляров.
Также реализована возможность у студентов и лекторов производиить операции сравнения по подсчету среднего бала.
